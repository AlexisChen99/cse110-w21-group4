start()

Precondition: there must be at least one task in order to start the timer. Starts the timer and decrements the timer's MM:SS every second. Will save the settings after every time the start button is pressed and hides the options to give the user a more focused screen. After every work phase, the timer will switch to displaying the break phase and its time. If the user completes all of their tasks while the timer is running, then the timer stops and a congratulations screen is shown.
